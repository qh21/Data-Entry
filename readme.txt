-The Database has two tables, one for the users and the other for the contacts.

-There are foreign key from contacts table to users table.

-The user can create accounts on the website.

-Each user can view, add, edit or delete their own contacts.

-The user can edit his account information and can delete his account.

-In the search area you can input phone number or contact's name it will automatically detect if it's a number or name to view the full informain of the contact.

-The user cannot use the webiste without login.
